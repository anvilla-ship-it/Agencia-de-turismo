# Llanquihue Tour

## Descripción

Sistema desarrollado en Java para representar personas relacionadas con una agencia de turismo mediante Programación Orientada a Objetos.

## Clases implementadas

### Direccion
Representa la dirección de una persona.

### Persona
Clase base que almacena información general.

### GuiaTuristico
Clase derivada de Persona que representa a un guía turístico.

## Paquetes

app
- Main.java

model
- Direccion.java
- Persona.java
- GuiaTuristico.java

## Conceptos POO utilizados

- Encapsulamiento
- Herencia
- Composición
- Constructores
- Getters y Setters
- Sobrescritura de toString()

## Ejecución

1. Abrir el proyecto en IntelliJ IDEA.
2. Ejecutar la clase Main.
3. Observar la salida por consola.
